# Relevant links

## Scientific work:

*DeGSA* (Dependence Graph Static Analyzer) -- static analysis tool for LuNA, dynamically typed declarative languaged based on concept of fragmented programming, uses enhanced AST and DDG models and different check algorithms to find semantic errors. Part of the ADAPT static analysis complex (registration), of which I am a developer also.
